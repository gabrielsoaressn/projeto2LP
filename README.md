# projeto2LP
##  projeto final da disciplina Linguagem de Programação
* O objetivo da disciplina foi aprender o paradigma de programação orientada a objeto.
* É um sistema que simula um o software de gerenciamento de insumos do Sistema Único de Saúde do Brasil, o SUS.
* A orientação a objeto é utilizada para que as classes vacina, EPI e medicamentos herdem da classe insumo.
* A maior dificuldade do projeto foi relativa ao uso de ponteiros e a escrita de arquivos.
